##Java文件名与Class的关系
1. 一个Java文件可以有多个class，有几个class就会编译出几个class文件，每个class都能有自己的main方法入口。
2. 但是一个java文件中只能有一个public class，且这个public class必须要与文件名相同。比如文件名为Demo.java,如果文件中有public class，那么这个类必须命名为Demo。
2. 并不一定需要public class。

##命名规范：
1. 变量名由大小写字母、数字、下划线(_)、美元符号($)组成；
2. 第一个不能是数字；
3. 不能用保留字命名；
4. 一般默认类名第一个字母大写，变量名第一个字母小写，常量名全部大写。采用驼峰法或者下划线连接法来组合命名。
